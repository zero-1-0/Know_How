C) CSS Files and Structure - top

Styles

We're using 10 CSS files in this template. we have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

bootstrap.min.css

This Stylesheet contains responsive grid system ,default element style based on bootstrap.It is bootstrap framework main css file

font-awesome.min.css

Base styles for FontAwesome icon fonts.

superslides.css

Styles for Full Screen slider (superslides).

slick.css

Styles for carousel slider (slick slider).

jquery.circliful.css

CDN Stylesheet file for skill counter circle.

animate.css

Styles for smooth animation.

queryLoader.css

Styles for preloader animation.

jquery.tosrus.all.css

Styles for gallery view plugin.

default-theme.css

This is default theme color file for this template.

style.css

Common css style and structures for this template.
